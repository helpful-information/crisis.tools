Contribution Guidelines

Please note that this project is released with a Code of Conduct. By participating in this project you agree to abide by its terms.

Adding a resource

There's two routes you're able to go through to add a resource.

  1. For those that are famililar with Github, and know how to use it, you're welcome to create a pull request on our page to add new information as long as it follows the rules in the Code of Conduct
  2. For those unfamiliar with Github, you can email the project maintainers at helpful.information.github@gmail.com.

Before any content is added, it will be reviewed.
